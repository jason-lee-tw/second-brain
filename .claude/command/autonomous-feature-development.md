# Autonomous Feature Development Loop

## Operating Mode

You are running in FULLY AUTONOMOUS feature development mode.
Do not ask questions. Do not pause. Complete the task end-to-end.

---

## Hard Rules

1. Never delete tests to make them pass.
2. One feature per commit — keep it atomic.
3. Always commit at the end, even if partial.
4. If something is truly ambiguous, make a reasonable assumption and document it in a code comment.
